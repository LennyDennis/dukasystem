                                                      Siti Aisyah Azman Peejay Alcantara 6 months ago edited

                                                      You have to change the directory of the file.
                                                      For example, include('functions.php').
                                                      If you save the functions.php in new folder, then you have write
                                                      include('folder/functions.php'). This is also apply for header('location: login.php'), href="style.css" and <img src="images/admin_profile.png">

                                                      The coding below is for those that save coding in htdocs only. If you save the coding in other folder in htdocs, then add your folder like I said at above.

                                                      <html>
                                                      <head>
                                                      <title>Home</title>
                                                      <link rel="stylesheet" type="text/css" href="style.css">
                                                      <style>
                                                      .header {
                                                      background: #003366;
                                                      }
                                                      button[name=register_btn] {
                                                      background: #003366;
                                                      }
                                                      </style>
